What blogging software?

One question I've recieved is "what blogging software are you using, your web pages don't look like anything I'm familiar with". Well... I wrote my own. It's really simple and does just what I want. With luck (mostly getting time) I'll open source it. The one big feature is that it will publish to a generic ftp-able website and it can work offline. My TiBook is often off the net and I wanted to be able to write entries whenever. I also frequently take myself off the net just so I can get things done.

[ Update: it is now open sourced at java.net ]
