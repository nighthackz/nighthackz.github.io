Swing on Steroids

There are a couple of cool new projects on dev.java.net: JDNC and JDIC.

JDNC, Java Desktop Networed Components is a set of extended versions of the more sophisticated Swing components (Table and TextArea) along with an XML based markup language to allow webstart and applet clients to be easily constructed.

JDIC, Java Desktop Integration Components provide more integration into the host desktop environment: integrating the native web browser, mail, registered document viewing applications and installation.
