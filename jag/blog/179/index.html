The race server is live...

The JavaOne slot car race server is open for business. If you're interesting in entering when you get to JavaOne, all of the materials are there. Rules. Code samples. NetBeans modules. Hints. It's all there, except that you can't actually enter yet. You can construct an entry, but you'll have to wait until you get to JavaOne to actually enter. We've been doing a dry run for the past few weeks. The whole rig seems to be in pretty good shape. It's pretty easy to get times under 30 seconds. Getting under 25 is challenging, but a few have done it. No one has broken 20 seconds yet: you can be the first.
