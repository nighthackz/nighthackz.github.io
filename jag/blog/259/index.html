Java Mobile&Embedded Developer Days

We're doing a special conference for mobile&embedded developers this coming week on Wednesday and Thursday at Sun's Santa Clara campus. The agenda is intensely technical. The conference is devoted solely to the technologies of mobile and embedded Java platforms and is targeted for application developers of intermediate and advanced skill levels, platform developers and technical personnel at tool vendors, OEM's and carriers. Registration is getting tight, but there's still some room. We're going to be streaming the video of the event at ustream.tv so that you don't have to miss out if you can't attend.
