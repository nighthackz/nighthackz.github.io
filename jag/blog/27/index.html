It talks!

Paul Lamere just checked in a cool feature for JNN: it talks! He tied FreeTTS (the Java-based text to speech engine at sourceforge) into JNN so that when new news is noticed, it speaks the headline. So that if you're a real news junkie, you can run JNN all the time and not have it take up any screen space. To enable speech, just install the FreeTTS libraries from sourceForge: the next time JNN is started, it will notice the existance of the libraries and start talking automatically.
